Using Blockchain for MEC Enhancement

Blockchain Minor Project

The objective of this project is to look for ways to improve the functionality of the Multi-Access Edge Computing, mainly the Network Function Virtualization (NFV) architecture, using Blockchain technology.

The project proposes a blockchain-enabled reputation system to identify the best possible VNF(Virtualized Network Function) provider to be chosen by the core server in the NFV (Network Function Virtualization) network.

This project was a part of my college's Minor Project in 3rd Year.
